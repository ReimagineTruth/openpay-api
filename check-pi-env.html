Configuration Instructions

⚠️ Important: Make sure your .env file contains the following Pi Network configuration: 
# Pi Network API Configuration
PI_API_KEY="your_actual_pi_api_key_here"
VITE_PI_WALLET_PRIVATE_SEED="S_YOUR_ACTUAL_WALLET_PRIVATE_SEED"
VITE_PI_SANDBOX="false"

Required Values

  • PI_API_KEY: Your Pi Network application API key
  • VITE_PI_WALLET_PRIVATE_SEED: Your Pi Network wallet private seed (must start with 'S')
  • VITE_PI_SANDBOX: Set to "false" for production, "true" for testnet
